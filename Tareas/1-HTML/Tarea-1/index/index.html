Conceptos Básicos

Definición

HTML, o Hypertext Markup Language, es un lenguaje de marcado que le dice a los navegadores web cómo estructurar las páginas. el HTML consiste en una serie de elementos

Elementos HTML

Las partes principales de un elemento son :

  • Etiqueta de un elemento
  • Contenido de un elemento
  • Etiqueta de cierre

Atributos

Los elementos también pueden tener atributos, estos brindan información extra sobre los elementos

Sus principales características son:

  • Espacio entre este y el nombre del elemento
  • Nombre del atributo
  • valor del atributo, rodeado por comillas

Anatomía de un documento HTML

<!DOCTYPE html>

Cuando el HTML llevaba poco tiempo (alrededor de 1991-1992), los doctypes servían como enlaces al conjunto de reglas que la página HTML debía seguir para que fuera considerado buen HTML.

En la actualidad se ignora y se considera un legado histórico que hay que incluir para que todo funcione correctamente <!DOCTYPE html>.

<html></html>

El elemento <html></html> envuelve toda la pagina, y se lo conoce como elemento raíz

<head></head>

Actúa como contenedor de parámetros que no serán visibles en la pagina, como por ejemplo keywords, charset, etc.

<body></body>

Contiene todo lo que quieras mostrar a los usuarios, textos, imágenes, etc.
